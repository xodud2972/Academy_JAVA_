[ 자바 1일차 ]
■ 자바 언어로 개발할 수 있는 분야
1)JavaSE ( 독립 실행형 응용프로그램 개발 )
2)JavaEE ( 웨 기반, 기업용 어플리케이션 개발 )
3)JavaME ( 임베디드 어플리케이션 개발 )
		우리의 경우 임베디드중에서도 안드로이드 스마트폰에서
		실행되는 안드로이드 플랫폼 개발을 학습함.
■ 환경변수 ( OS에 등록되는 전역변수 )
1) 명칭이 정해진 것 : 
path : 실행할 프로그램의 경로를 담고 있는 변수
java_home : java 개발 환경의 루트 디렉토리 경로를 담고 있는 변수

2) echo 문자열 : 해당 문자열 콘솔에 출력하는 기능
		단, 출력대상 문자열이 환경변수(%%)인경우 그 값을 출력해준다.
3) 사용시 장점 : 경로를 일일이 명시하지 않아도 됨
■ 자바의 컴파일/실행 명령어
1) 컴파일 : javac.exe 대상 클래스명.java
2) 실행 : java.exe 대상 클래스명 

■ 자바의 자료형
1) 총 3가지 문자(char), 숫자(정수와 실수), 논리값(boolean)
2) 용량	정수 1, 2, 4, 8 byte
		실수 4, 8 byte
3) char 형은 사실 문자코드체계의 정수값으로 대체될 예정이기 때문에 사실 숫자형이다.
	ex) char c = 65; 가능하다, 또한 컴파일러의 수고로움을 덜어줌...
