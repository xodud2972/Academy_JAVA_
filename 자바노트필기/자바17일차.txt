[VO , DTO]

VO : Value Object
	로직을 작성하기 위한 	객체가 용도가 아니라, 오직 데이터만을 담기위한 목적으로 정의된 객체

DTO : Data Transfer Object 
	데이터 전달을 목적으로 하는객체의 의미로써, VO와 동일

[DB분야 vs OOP분야]

				DB분야			OOP분야	
------------------------------------------------------------------
현실의 객체표현틀		 Table			class
틀로부터 생성된 객체		record			인스턴스
객체의 속성			column			property
